# ЁЯУЦ About

This project is a **Slicing Figma to Website Challenge**.

The goal is to convert a modern portfolio UI from Figma into a production-ready website while maintaining:

- Pixel Perfect Layout
- Responsive Design
- Clean Code Architecture
- Smooth Animation
- Reusable Components
- SEO Friendly
- Deploy Ready

# ЁЯУж Installation

Clone repository

```bash
git clone https://github.com/JamesBangguna/frontend-portfolio.git
```

Go to project

```bash
cd frontend-portfolio
```

Install dependencies

```bash
npm install
```

Run development

```bash
npm run dev
```

Open

```
http://localhost:3000
```
